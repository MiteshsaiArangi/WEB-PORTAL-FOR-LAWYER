# ⚖️ CaseFlow – Smart Web Portal for Lawyers

CaseFlow is a full-stack web application designed to simplify legal service management by connecting clients with verified lawyers through a secure, modern, and user-friendly platform. The system enables users to search for lawyers, manage legal profiles, authenticate securely using JWT, and communicate efficiently, while providing dedicated dashboards for lawyers and administrators.

## ✨ Features

### 👤 Client
- User Registration & Login
- Browse Verified Lawyers
- Search Lawyers by Name, City, State, and Specialization
- View Lawyer Profiles
- Contact Lawyers
- Secure JWT Authentication

### 👨‍⚖️ Lawyer
- Lawyer Registration
- Lawyer Dashboard
- Manage Professional Profile
- Update Availability
- View Client Requests

### 👨‍💼 Admin
- Admin Dashboard
- Verify Lawyer Profiles
- Manage Lawyers
- Manage Users
- Role-Based Access Control

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/MiteshsaiArangi/CaseFlow.git
```

### Backend Setup

```bash
cd backend
```

Run the application

```bash
./mvnw spring-boot:run
```

or

```bash
mvn spring-boot:run
```

Backend URL

```
http://localhost:8080
```

---

### Frontend Setup

Navigate to the frontend folder and open

```
index.html
```

using Live Server or any static web server.
